## **Common AI Limitations**

### **What AI is NOT Good At (Yet):**

‚ùå **Perfect Math** - Can make calculation errors (verify important math)
‚ùå **Counting** - Often miscounts words, characters, items in lists
‚ùå **Current Events** - Training data has cutoff dates (unless it has web search)
‚ùå **Remembering Long Conversations** - May forget details from 50+ messages ago
‚ùå **Being Original** - Combines existing patterns, doesn't truly create new ideas
‚ùå **Understanding Context Perfectly** - Can misinterpret nuance
‚ùå **Avoiding All Hallucinations** - Will sometimes make up facts confidently
‚ùå **Replacing Professionals** - Not a substitute for doctors, lawyers, accountants

### **Red Flags for Hallucinations:**

üö© Suspiciously specific statistics without sources
üö© Citations that sound real but you can't find
üö© "Studies show..." without naming the study
üö© Historical "facts" that seem too convenient
üö© Exact numbers when asked for approximations
üö© Perfect answers to very niche questions

**Always verify important information!**

## **Useful Prompt Templates**

### **For Better Research:**
```
"I need to understand [topic]. Explain it in simple terms,
then give me 3 key takeaways, and suggest 3 follow-up
questions I should ask."
```

### **For Decision Making:**
```
"I need to decide between [Option A] and [Option B].
Create a comparison table with pros, cons, costs, and
best use cases for each."
```

### **For Learning:**
```
"I want to learn [skill/topic]. I'm a complete beginner.
Create a 30-day learning plan with daily tasks that take
30 minutes or less."
```

### **For Writing:**
```
"Help me write a [type of document] for [audience] about
[topic]. The tone should be [professional/casual/etc.].
Draft three different opening paragraphs for me to choose from."
```

### **For Problem Solving:**
```
"I'm facing this problem: [describe]. Help me think through
this step-by-step. What are my options? What are the
pros and cons of each?"
```

### **For Improvement:**
```
"Here's my [email/document/plan]: [paste it]
How can I make this better? Give me specific suggestions."
```

## **Troubleshooting Common Issues**

**"AI is at capacity" / Can't access**
- Try a different AI provider
- Wait and try again later
- Consider upgrading to paid tier

**"Response seems wrong"**
- Verify the information (don't trust blindly)
- Try rephrasing your question
- Ask AI to show its reasoning
- Try a different AI

**"Not getting good results"**
- Be more specific in your prompt
- Add context about your situation
- Ask AI to improve your prompt (meta move)
- Try the 4 rules: specific, context, format, role

**"Forgot what we discussed earlier"**
- AI has context limits
- Start a new conversation for new topics
- Summarize key points if continuing long conversation

**"Can't create account"**
- Check email spam folder for verification
- Try different email provider
- Clear browser cache
- Contact support

## **For Instructors: Teaching Tips**

### **Common Student Concerns:**

1. **"This is too complicated"**
   - Start with one simple task
   - Show immediate value
   - Build confidence gradually

2. **"I'm afraid of breaking something"**
   - Reassure: You can't break AI by asking questions
   - Worst case: Bad answer, just ignore it
   - Experimentation is encouraged

3. **"I don't trust it"**
   - Good! Healthy skepticism is valuable
   - Teach verification methods
   - Show examples of mistakes

4. **"Will this replace my job?"**
   - AI is a tool to enhance your work
   - Those who use AI will have advantage over those who don't
   - It's about augmentation, not replacement
